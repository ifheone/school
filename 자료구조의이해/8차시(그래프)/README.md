# 그래프 1

- 연결할 객체를 나타내는 정점과 객체를 연결하는 간선 집합으로 구성된 그래프 G를 G=(V,E)로 정의한다.
- 그래프는 간선의 방향성 유무에 따라서 방향 그래프와 무방향 그래프가 있고 연결 정도에 따라 연결그래프와 단절그래프, 완전 그래프가 있다. 그리고 가중치를 가진 간선으로 이루어진 가중치 그래프가 있다.
